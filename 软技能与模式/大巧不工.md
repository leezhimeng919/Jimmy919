# 原型稿
## UCD 
- User-Centered Design以用户为中心的设计原则
- 关注用户
    + 用户群体分析
    + 人物角色分析：人物角色是对网站目标群体真实特征的勾勒
- 用户调研
    + 问卷调查
        * 真诚待人
        * 主动适应，适当应道
        * 不忽略细节
    + 可用性测试
        * 邀请一些特定的用户对将要发布的功能进行试用
- 更含蓄的方式
    + 服务器端数据
        * 服务器日志文件
        * 网络监视器收集到的数据
        * 服务器端存储的其他类型数据
    + 客户端数据
        * 用户鼠标移动轨迹等
        * 脚本技术、插件技术、网页跟踪、Java applet收集客户端数据
    + 数据挖掘
